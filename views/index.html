Download links & Setup instructions for IoT Workshop


  • info_outlineRequirements
    In order for this workshop to be successful, we need the following things and few setup configurations.

    1. Arduino IDE installed and configured to connect devices with USB
    2. Developer Edition Salesforce Org
    3. A Heroku App with MQTT server as we need it communicate with the ESP32 module.

    Please go through each tab as it has detailed information on setup. If you face any problem, don't hesitate to contact me. Tweet to @NktTanjea
  • laptopArduino IDE setup
    1. Goto Arduino IDE downloads page and install the approriate version.
    2. In order to connect to our ESP32 module via USB port, we need extra drivers / boards. Check out the following blog articles for respective OS'
      1. Mac / Linux users. (Note: You many need to install ESP32 CP210x USB to UART Bridge VCP Drivers. It's mentioned in the blog but can be easily overlooked hence writing here.)
      2. Windows
    3. We would need to install additional libraries in order for our code to execute properly. Checkout this link to learn more about it. Please download and install the following libraries.
      1. PubSub Client
      2. Arduino Json
  • filter_dramaSalesforce setup
    1. We need a Developer Edition org with DevHub enabled. Since Salesforce has disabled IoT cloud by default, we need a Scratch Org. You can download the ScractOrg definition file here.
    2. Please create a Connected App in the Scratch Org with OAuth and Device Flow Enabled. We would need the client_id to register our device.
  • codeHeroku App + Sample Code
    1. Please click the button in order to deploy a copy of this supporting webapp. It starts a cloudMQtt server for you. Alternatively, you can also fork this repo and do it yourself.
    2. Goto the YD19-Sketches repo and please download the sketches.
